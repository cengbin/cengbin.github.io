# 加密算法示例

本目录包含三种常见加密算法的 Node.js 示例代码。

## 安装依赖

```bash
npm install crypto-js jsencrypt
```

## 示例文件

### 1. 消息摘要算法 (1-message-digest.js)

演示了使用 `crypto-js` 库实现的哈希算法：
- MD5
- SHA-1
- SHA-256
- SHA-512
- HMAC-SHA256

**运行:**
```bash
node test/1-message-digest.js
```

**特点:**
- 单向不可逆
- 固定长度输出
- 相同输入产生相同输出
- 微小改变产生完全不同的哈希值

### 2. 对称加密算法 (2-symmetric-encryption.js)

演示了使用 `crypto-js` 库实现的对称加密：
- AES 加密/解密
- DES 加密/解密
- Triple DES (3DES) 加密/解密
- 自定义 IV 和加密模式

**运行:**
```bash
node test/2-symmetric-encryption.js
```

**特点:**
- 加密和解密使用相同的密钥
- 速度快，适合大量数据加密
- 需要安全地共享密钥

### 3. 非对称加密算法 (3-asymmetric-encryption.js)

演示了使用 `jsencrypt` 和 Node.js `crypto` 模块实现的 RSA 加密：
- 公钥加密，私钥解密
- 数字签名和验证
- 密钥对生成
- 实用类封装

**运行:**
```bash
node test/3-asymmetric-encryption.js
```

**特点:**
- 使用公钥加密，私钥解密
- 使用私钥签名，公钥验证
- 无需共享私钥
- 速度较慢，适合小量数据或密钥交换

## 使用场景对比

| 算法类型 | 使用场景 | 示例 |
|---------|---------|------|
| 消息摘要 | 密码存储、数据完整性校验、数字指纹 | 用户密码哈希、文件校验 |
| 对称加密 | 大量数据加密、会话加密 | 文件加密、数据库字段加密 |
| 非对称加密 | 密钥交换、数字签名、身份认证 | HTTPS、数字证书、JWT |

## 安全建议

1. **消息摘要**: 使用 SHA-256 或更高版本，避免使用 MD5 和 SHA-1
2. **对称加密**: 使用 AES-256，确保密钥安全存储
3. **非对称加密**: 使用至少 2048 位的 RSA 密钥，私钥必须安全保管

## 混合加密方案

实际应用中常常结合使用：
1. 使用非对称加密交换对称密钥
2. 使用对称加密加密实际数据
3. 使用消息摘要验证数据完整性
